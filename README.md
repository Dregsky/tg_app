# tg_app
===========BALANÇO==============
Os dados do balanco estão divididos por 1000.
Por isso número de ações também está.
Consequentemente os números das planilhas em empresa_dados_mes_ano 
também estão,ou seja, o valor real é multiplicado por 10³

Os balanços são tirados do site
http://www.fundamentus.com.br/detalhes.php
a condição para que a empresa seja "aceita" é que todos os indicadores
fundamentalistas, estejam apresentados e a data da última cotação seja 
pelo maior ou igual 30/06/2017

O Balanço deve ter a data maxima de 30/06/2017, caso tenha informações
de superior a essa data, apagar coluna.

O número de ações deve ser adicionado na aba Bal. Patrim. na celula A2,
sendo que esse valor deve ser divido por 1000.

==========COTAÇÃO===========
As cotações são tiradas do site:
https://br.financas.yahoo.com/lookup

O período é 01/11/2012(equivale a 31/10/2012) a 01/07/2017(equivale 30/06/2017)
frequência mensal
A planilha baixada no site vem em formato csv
Esses dados devem ser exportados e ordenado do maior pro menor 01-07-2017 a 01-11-2012
e salvo como xlsx
